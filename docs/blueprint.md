# **App Name**: AdaptiveMind AI

## Core Features:

- User Authentication: Secure user login and signup with password protection and social login options (Google, GitHub).
- Personalized Learning Paths: AI-powered adaptive learning tool generates personalized learning paths based on user proficiency, goals, and preferred learning styles.
- Content Recommendations: Suggests relevant learning resources (articles, videos, exercises) based on user's learning path and progress.
- Progress Tracking: Monitors user progress, provides performance insights, and identifies areas for improvement.
- Account Management: Allow users to edit user profile information

## Style Guidelines:

- Primary color: Neon cyan (#00E5FF) for main elements like the logo outline, button glows, and highlights.
- Background color: Deep black (#000000) to enhance the neon glow and create an immersive feel.
- Accent color: Neon purple (#B500FF) for secondary highlights, button glows, and social login icons.
- Body and headline font: 'Space Grotesk' sans-serif font for a modern, futuristic look. Use 'Inter' for body if longer text is needed.
- Abstract brain or neural circuit icon rendered with glowing neon cyan and purple gradient lines for the logo.
- Full-screen layout with centered logo section, vertically stacked glassmorphic buttons, and subtle footer.
- Subtle animated background with dark particle motion and glow pulse on button hover/tap.
